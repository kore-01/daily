## 🔧 部署步骤

### 1. 服务器准备

```bash
# 更新系统
apt update && apt upgrade -y

# 安装基础依赖
apt install -y python3 python3-pip python3-venv git curl wget \
    build-essential libssl-dev zlib1g-dev libbz2-dev \
    libreadline-dev libsqlite3-dev llvm libncurses5-dev \
    libncursesw5-dev xz-utils tk-dev libffi-dev liblzma-dev

# 创建项目目录
mkdir -p /opt/daily_stock
cd /opt/daily_stock
```

### 2. Python环境配置

```bash
# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 升级pip
pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt
```

### 3. 前端构建

```bash
# 在本地或构建服务器上执行
cd apps/dsa-web
npm install
npm run build

# 构建输出到 static/ 目录
# 将 static/ 上传到服务器 /opt/daily_stock/static/
```

### 4. 环境变量配置

创建 `/opt/daily_stock/.env`：

```env
# 股票列表 (逗号分隔)
STOCK_LIST=300766,000301,920726,000026,000617

# Tushare Token
# 获取地址: https://tushare.pro/register
TUSHARE_TOKEN=your_tushare_token_here

# AI模型配置
# 选项: deepseek-v3, minimax, gemini-1.5-pro, gemini-2.0-flash
MODEL=minimax

# DeepSeek配置
DEEPSEEK_API_KEY=your_deepseek_key
DEEPSEEK_BASE_URL=https://api.deepseek.com

# MiniMax配置 (推荐)
MINIMAX_API_KEY=your_minimax_key
MINIMAX_GROUP_ID=your_group_id
MINIMAX_MODEL=MiniMax-Text-01

# Gemini配置
GOOGLE_API_KEY=your_google_key

# Tavily搜索API Keys (多个密钥轮询使用)
# 获取地址: https://tavily.com/
TAVILY_API_KEYS=tvly-dev-xxxxx,tvly-dev-yyyyy,tvly-dev-zzzzz

# 飞书Webhook配置
# 获取地址: 飞书群设置 -> 添加机器人
FEISHU_WEBHOOK_URL=https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx
FEISHU_SECRET=your_feishu_secret

# 功能开关
ENABLE_CHIP_DISTRIBUTION=true
ENABLE_FEISHU_NOTIFICATION=true
ENABLE_TAVILY_SEARCH=true

# 时区设置
TZ=Asia/Shanghai

# 日志级别
LOG_LEVEL=INFO
```

### 5. Systemd服务配置

创建 `/etc/systemd/system/daily-stock.service`：

```ini
[Unit]
Description=Daily Stock Analysis Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/daily_stock
Environment=PYTHONPATH=/opt/daily_stock
Environment=TZ=Asia/Shanghai

# 加载环境变量
EnvironmentFile=/opt/daily_stock/.env

# 启动命令
ExecStart=/opt/daily_stock/venv/bin/python main.py --webui-only

# 重启策略
Restart=always
RestartSec=10

# 日志
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```

启用服务：

```bash
systemctl daemon-reload
systemctl enable daily-stock
systemctl start daily-stock
systemctl status daily-stock
```

### 6. 定时任务配置

```bash
# 编辑crontab
crontab -e

# 添加以下内容（北京时间，周一到周五）
SHELL=/bin/bash
PATH=/usr/local/sbin:/usr/local/bin:/sbin:/bin:/usr/sbin:/usr/bin

# 早盘分析 (09:40)
40 1 * * 1-5 cd /opt/daily_stock && source venv/bin/activate && python main.py >> /opt/daily_stock/logs/cron.log 2>&1

# 盘中分析 (10:30)
30 2 * * 1-5 cd /opt/daily_stock && source venv/bin/activate && python main.py >> /opt/daily_stock/logs/cron.log 2>&1

# 午后分析 (13:30)
30 5 * * 1-5 cd /opt/daily_stock && source venv/bin/activate && python main.py >> /opt/daily_stock/logs/cron.log 2>&1

# 尾盘分析 (14:20)
20 6 * * 1-5 cd /opt/daily_stock && source venv/bin/activate && python main.py >> /opt/daily_stock/logs/cron.log 2>&1
```

### 7. 防火墙配置

```bash
# 开放8000端口
ufw allow 8000/tcp
ufw reload

# 或使用iptables
iptables -A INPUT -p tcp --dport 8000 -j ACCEPT
```

## 📊 数据源配置

### Tushare (股票数据)

1. 注册账号: https://tushare.pro/register
2. 获取Token: 个人中心 -> 接口TOKEN
3. 配置到 `.env`

```env
TUSHARE_TOKEN=your_token_here
```

### Tavily (AI搜索)

1. 注册账号: https://tavily.com/
2. 获取多个API Key（免费版每月1000次）
3. 配置到 `.env`，支持多Key轮询

```env
TAVILY_API_KEYS=key1,key2,key3,key4,key5
```

## 🚨 故障排查

### 常见问题

#### 1. 服务无法启动

```bash
# 检查Python环境
source /opt/daily_stock/venv/bin/activate
python --version  # 应为3.11+

# 检查依赖
pip list | grep -E "fastapi|uvicorn|tushare"

# 检查端口占用
ss -tlnp | grep 8000
```

#### 2. Tavily API错误

```bash
# 测试API Key
python3 -c "
from api.tavily_search import TavilySearch
search = TavilySearch()
result = search.search('测试')
print(result)
"
```

#### 3. Tushare数据获取失败

```bash
# 测试Token
python3 -c "
import tushare as ts
ts.set_token('your_token')
pro = ts.pro_api()
df = pro.daily(ts_code='000001.SZ', start_date='20260101')
print(df.head())
"
```

#### 4. 前端不显示

```bash
# 检查静态文件
ls -la /opt/daily_stock/static/

# 检查目录权限
chown -R root:root /opt/daily_stock/static
```
